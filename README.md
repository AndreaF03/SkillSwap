# SkillSwap

A community-driven platform where users can teach, learn, and exchange skills through mentorship, collaboration, and AI-powered recommendations.

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Django REST Framework

### Database
- PostgreSQL

### Deployment
- Vercel
- Render
- Neon

## Features

- Authentication
- User Profiles
- Skill Matching
- Exchange Requests
- Reviews & Ratings
- Credit Economy
- Learning Pods
- AI Recommendations
